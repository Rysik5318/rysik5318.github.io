3.1 Выравнивание с FlexBox Layout

Создадим контейнер с 5 широкими бловками.
Листинг 3.1: body (index.html)
<body>
<div class="outer">
<div class="inner">1</div>
<div class="inner">2</div>
<div class="inner">3</div>
<div class="inner">4</div>
<div class="inner">5</div>
 </div>
 </body>
Листинг 3.2: style.css
html, body {
/* По умолчанию высота html и body определяется контентом. */
 height: 100%
 }
body {
/* В большинстве браузеров body имеет margin равный 8px */
 margin: 0;
 }
.inner {
background-color: red;
 margin: 15px;
 border-radius: 5px;
 width: 300px;
 font-size: 72 px;
 text-align: center;
 color: white;
 }
.outer {
}

Как и положена обычным блокам, каждый из них занимает отдельную строчку.
Добавим свойство display в значении flex к классу outer.
Блоки сжались и выстроились на одну строку.
Уменьшите ширину окна браузера — блоки продолжат сжиматься, так как по умолчанию элементы внутри flex-контейнера располагаются на одной строке вне зависимости от заданных размеров.
В следущих подразделах мы познакомимся с некоторыми методами выравнивания при помощи FlexBox.
Для лучшего понимания рекомендуется так же экспериментировать с резамерами окна.

3.1.1 flex-wrap

Чтобы сохранять размеры элементов внутри контейнера и при необходимостри переносить их на следующую строчку, использользуют свойство flex-wrap в значении wrap
Листинг 3.3: .outer (style.css)
.outer {
display: flex;
 flex-wrap: wrap;
 }

Пример:

(Уменьшенная версия)
1
2
3
4
5

3.1.2 justify-content

Мы уже использовали свойство justify-content со значением center, центрирующее элементы по горизонтали.
Листинг 3.4: .outer (style.css)
.outer {
display: flex;
 flex-wrap: wrap;
 justify-content: center;
 }

Пример:

(Уменьшенная версия)
1
2
3
4
5
У этого свойства могут быть и другие значения
  • flex-start, используемое по умолчанию и прижимающие элементы к началу строки;
  • flex-end, прижимающее элементы к концу строки;
  • space-around, располагающее элементы равномерно на строке с равными отступами с обеих сторон;
  • и space-between, похожее на space-around, отличающееся лишь отсутвием отступов у крайних элементов.
