Гайд по пользованию команд в SCP:SL

(Только для Админов)

htag/stagСкрыть/Показать ваш собственный префикс

bc/alert [Время (в сек)] ТекстОтправляет всем текст наверху экрана.
Видео с Youtube

@ ТекстОтправляет текст в админ чат наверху экрана
Видео с Youtube

ban [Айди/Имя игрока] [Время(h,d,y)] ТекстБанит игрока на время на сервере

speak/icomС помощью этой команды можно говорить в интеркоме на расстоянии.
Видео с Youtube

cassie ТекстВыдаёт оповещение с вашим текстом.
Подробный гайд - *клик*
Видео с Youtube

cassie_sl Текст

Выдаёт оповещение с вашим текстом, но удаляет начальное оповещения о объявлении.

Подробный гайд - *клик*
Видео с Youtube

kill [Айди/Имя игрока]Убивает игрока.
Видео с Youtube

pinv [Айди/Имя игрока]Кидает список предметов его инвентаре.
Видео с Youtube

speak/icomС помощью этой команды можно говорить в интеркоме на расстоянии. (Примечание: Если какой-то игрок тоже напишет speak с нужными правами, а вы разговариваете с помощью него, то вас будет слышно как обычно)
Видео с Youtube

hp [Айди/Имя игрока]Выдаёт хп заданному игроку.
