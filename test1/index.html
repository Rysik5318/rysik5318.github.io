Гайд по пользованию команд в SCP:SL

(Только для Админов)

Команда: bc или alert Время (в сек) Текст

Отправляет всем текст наверху экрана.
Видео с Youtube

Команда: @ Текст

Отправляет текст в админ чат наверху экрана

Команда: cassie Текст

Выдаёт оповещение с вашим текстом.
Подробный гайд - *клик*
Видео с Youtube

Команда: cassie_sl Текст

Выдаёт оповещение с вашим текстом, но удаляет начальное оповещения о объявлении.
Подробный гайд - *клик*
Видео с Youtube

Команда: kill Айди/Имя игрока

Убивает игрока.

Команда: pinv Айди/Имя игрока

Кидает список предметов его инвентаре.
Видео с Youtube

Команда: speak

Спомощью этой команды можно говорить в интеркоме на расстоянии. (Примечание: Если какой-то игрок тоже напишет speak с нужными правами, а вы разговариваете с помощью него, то вас будет слышно как обычно)
Видео с Youtube
